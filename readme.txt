Automatically adds secure attributes to external links in post content.

== Description ==

Smart External Links Manager detects external links in post content and adds:

* target="_blank"
* rel="noopener noreferrer"

Internal links, anchor links, mailto, and tel links are ignored.
This plugin is intentionally lightweight and works out of the box without configuration.

== Installation ==

1. Upload the plugin folder to /wp-content/plugins/
2. Activate the plugin through the Plugins menu
